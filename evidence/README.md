# Runtime Evidence

- `runtime-dashboard.png` - live Phase V runtime dashboard with two attached
  products, two sessions, two routed dispatches, and zero failures.
- `runtime-openapi.png` - live OpenAPI 3.1 explorer showing the versioned
  session, context, attachment, intent, and dispatch APIs.
- `runtime-health.png` - live health response.
- `runtime-intents.png` - live intent registry populated only by product
  manifests.
- `mitra-companion-runtime-demo.mp4` - 42-second animated proof walkthrough
  covering GitHub commit proof, product self-attachment, integration contracts,
  runtime simulation, tests, validation, and review packet evidence. This is a
  continuous screencast-style video, not a screenshot montage.
- `proof-video-metadata.json` - generation metadata for the proof video,
  including commit, duration, resolution, API path count, schema count, and test
  result cited in the video.
- `demo-transcript.json` - executable two-product flow with explicit context
  transfer and product-isolation proof.

Screenshots were captured from `http://127.0.0.1:8090` using the persisted demo
database generated by `scripts/run_demo.py`. The proof video is generated by
`scripts/make_proof_video.py` from checked-in runtime facts and validation
metadata.
